Foundation

First Principles: Physics & Electronics

Circuit theory, passive/active components, digital logic. Build intuition before code.

Circuit Theory & Core Concepts
▶
The Organic Chemistry Tutor — Electronics Playlist
Crystal clear explanations of Ohm's Law, KVL, KCL, RC circuits. The best "first pass" on YouTube. Zero fluff.
YouTube
🌐
Electronics Tutorials (electronics-tutorials.ws)
The single best reference site for passive/active components. Covers BJTs, MOSFETs, diodes, RC time constants, power supplies. Bookmark it permanently.
Website
📄
All About Circuits — Free Textbook (6 vols)
Full-length electronics textbook series, completely free online. DC, AC, Semiconductors, Digital, Reference. Treat this as your course textbook.
Free PDF
🌐
CircuitLab — Ultimate Electronics Textbook
Interactive, browser-based textbook. Simulate circuits as you read. Exceptional for RC time constants and op-amp sections.
Website
University-Level Courses (Free)
🎓
MIT 6.002 — Circuits & Electronics (OCW)
The gold standard. Prof. Anant Agarwal's full course: lectures, problem sets, exams, all free. Combines RLC theory with MOSFETs and digital logic. This is a real EE course, not a hobbyist tutorial.
Free Course
▶
Neso Academy — Network Theory / Electronic Devices
Full GATE-level coverage of KVL, KCL, Thevenin, Norton, transistors. Methodical, exam-grade explanations. Excellent for self-testing.
YouTube
Digital Logic
▶
Ben Eater — YouTube Channel
Builds an 8-bit computer on breadboards from scratch: logic gates → flip flops → ALU → CPU. Nobody in the world teaches digital logic to hardware like Ben Eater. Required watching.
YouTube
⚙
Nand Game
Browser game. Build a full computer from NAND gates up. Covers AND, OR, XOR, flip-flops, ALU, CPU in an interactive format. Surprisingly addictive.
Tool / Game
Simulation Tools
⚙
Falstad Circuit Simulator
Free, browser-based SPICE simulator. Watch electrons flow in real time. Essential for understanding RC charging curves, transistor switching, and flyback diodes. No install required.
Free Tool
⚙
LTSpice (Analog Devices)
Industry-standard SPICE simulator. Free. Use this to simulate buck/boost converters and MOSFET gate-drive circuits before touching hardware.
Free Tool

Introduction to Arduino — Playlist
Use this as a gentle on-ramp before stripping away the abstractions in Phase 2. Understand what the libraries are doing before you replace them with register-level code.

Milestone

Build an NE555 astable multivibrator on a breadboard using only a datasheet — no tutorials. Calculate the resistors yourself. Oscilloscope the output. When it matches your math, Phase 1 is done.

03
Concurrency

Real-Time Operating Systems

Tasks, scheduling, mutexes, queues. Deterministic execution for complex systems.

Core RTOS Concepts
🌐
FreeRTOS Official Beginner's Guide
The primary source. Covers task states, scheduler, context switching, queues, mutexes, semaphores, and priority inversion. Dense but authoritative. Read it twice.
Website
▶
DigiKey Electronics — FreeRTOS Series
Shawn Hymel's FreeRTOS tutorial series is the best practical introduction on YouTube. Implements tasks, queues, and mutexes on real hardware with clean explanations of the "why".
YouTube
🌐
Embetronicx — FreeRTOS Tutorial Series
Implementation-focused tutorials for FreeRTOS on STM32: task creation, stack sizing, message queues, binary semaphores, event groups. Paired with code examples.
Website
Reference Books (Free PDFs)
📄
Mastering the FreeRTOS Kernel — Richard Barry (Free PDF)
Written by the FreeRTOS creator. Free official PDF. The definitive textbook. Covers the internals you won't get from YouTube: heap management models, tick interrupts, idle task, software timers.
Free PDF
📄
Operating Systems: Three Easy Pieces (OSTEP) — Free
The best OS fundamentals book, free online. Not RTOS-specific but essential for understanding scheduling, synchronization, and memory that underpins all RTOS concepts.
Free PDF
Milestone

On an ESP32: Task A reads a sensor at 100 Hz, Task B runs a PID loop at 50 Hz, Task C updates an OLED at 10 Hz. Data passes between tasks via FreeRTOS queues. No global variables. No race conditions. When this runs cleanly, you've internalized RTOS.

04
Linux Brain

Embedded Linux & Edge Computing

Kernel space, device drivers, custom distros, Python ML pipelines on SBCs.

The Standard Textbook
📚
Mastering Embedded Linux Programming — Chris Simmonds (Packt)
The industry-standard book for embedded Linux. Covers bootloaders (U-Boot), kernel configuration, Yocto/Buildroot, device drivers, real-time Linux. The bible for Phase 4. Affordable on Packt sales.
Book (Paid)
Free Online Resources
🌐
Raspberry Pi Headless Setup — VS Code & SSH (freeCodeCamp)
The first thing you should do when you get a Raspberry Pi. No monitor, no keyboard — just SSH and VS Code Remote. This is how professional embedded Linux dev actually works.
Website
📄
Bootlin — Free Embedded Linux Training Materials
Bootlin is the most respected embedded Linux consultancy in the world. They release their full training slides free: Embedded Linux, Buildroot, Yocto, Linux Kernel, Device Drivers. Hundreds of slides, all free. This is a hidden gem.
Free PDFs
🌐
Yocto Project Official Documentation
Start with the "Quick Build" guide to get your first image running, then the "Development Tasks Manual". The full reference is dense but necessary once you're building custom layers.
Website
🌐
Buildroot Official Docs + User Manual
Start here before Yocto. Buildroot is simpler and faster. The manual is clear and well-organized. Build a minimal Raspberry Pi image in a few hours.
Website
▶
Linux Device Drivers — YouTube (various series)
Search "Linux device driver tutorial" on YouTube. Multiple series covering char drivers, platform drivers, I2C/SPI kernel drivers. Pair with the Linux Kernel Module Programming Guide (free PDF, available at tldp.org).
YouTube
Milestone

Python script on a Raspberry Pi: OpenCV detects a colored ball, calculates its pixel coordinates, sends X/Y over UART to an STM32 which moves two servos to track it. This integrates Phase 2, 3, and 4 into one system.

05
Robotics Core

ROS 2 & Kinematics

Nodes, topics, TF2, forward/inverse kinematics, PID control, manipulators.

ROS 2 — Start Here
🌐
ROS 2 Official Tutorials (Jazzy)
The official tutorials are actually excellent: Beginner CLI tools → Client Libraries → Intermediate → Advanced. Complete them sequentially. Don't skip the Turtlesim section — it teaches the graph architecture you'll use on real hardware.
Website
▶
Open Robotics — 120+ Free ROS2 Training Sessions (YouTube)
120+ hours of structured ROS1 and ROS2 training, now free on YouTube. Beginner → Intermediate → Advanced. Includes slides and GitHub repos. Unbeatable value — this was commercial training.
YouTube
🌐
The Robotics Back-End Blog + YouTube
Edouard Renard's blog and channel. Best practical tutorials for ROS2 nodes, topics, services, parameters, and launch files in Python and C++. Clear, no-fluff writing.
Website
🎓
The Construct (Robot Ignite Academy)
Browser-based ROS environment — no local install needed. Free tier covers ROS2 basics with real robot simulation. Invaluable if you don't have a Linux machine ready yet.
Course (Free Tier)
Kinematics & Control
▶
Modern Robotics — Kevin Lynch (YouTube + Book)
The textbook for serious robotics kinematics. The companion YouTube playlist covers Denavit-Hartenberg parameters, forward/inverse kinematics, Jacobians. The full PDF textbook is free on the course website.
YouTube
▶
Brian Douglas — PID Control Series
The clearest PID explanation on YouTube. Control system intuition before math. Covers P, I, D terms, tuning methods, anti-windup. Brian's whole control theory channel is worth watching.
YouTube
Linear Algebra (Required for Kinematics)
▶
3Blue1Brown — Essence of Linear Algebra
16 videos. Watch this before touching kinematics. Rotation matrices, dot products, eigenvectors — you will never understand them as intuitively anywhere else. Non-negotiable.
YouTube
Milestone

3-DOF robotic arm: ROS 2 computes inverse kinematics from a target XYZ coordinate, publishes joint angles to topics, which an RTOS microcontroller subscribes to and physically moves servos to. Hardware + software fully integrated.

06
Perception

Sensor Fusion, SLAM & Navigation

Extended Kalman Filters, LIDAR, depth cameras, autonomous mapping and navigation.

Sensor Fusion & Kalman Filters
⎇
Kalman and Bayesian Filters in Python — Roger Labbe (Free Book)
Jupyter notebook textbook. The most accessible, visual introduction to Kalman Filters, Extended Kalman Filters, and Unscented KF in existence. Free on GitHub. Work through the notebooks chapter by chapter.
Free Book
▶
Michel van Biezen — Extended Kalman Filter Series
Short, mathematical derivations. Dense but precise. Pair with Labbe's visual notebooks for intuition + rigor.
YouTube
SLAM & ROS 2 Nav2
🌐
Nav2 Official Docs & Tutorials
The ROS 2 navigation stack. Covers SLAM Toolbox integration, costmaps, path planners, recovery behaviors. Work through the official tutorials in Gazebo simulation first before putting it on hardware.
Website
▶
Articulated Robotics — YouTube
Josh Newans builds a full ROS2 robot from scratch on YouTube — URDF, Gazebo, ros2_control, Nav2, SLAM. The most complete "real robot" ROS2 series available. Follow along and build yours at the same time.
YouTube
⎇
SLAM Toolbox — GitHub
The recommended SLAM package for ROS 2. Open-source, well-maintained, supports 2D and 3D mapping. Read the README thoroughly — it explains when to use each mode.
GitHub
Milestone

Differential-drive rover with an RPLidar A1 on a Jetson Nano. Uses ROS 2 + Nav2 + SLAM Toolbox to autonomously map a room and navigate to goal coordinates while avoiding dynamic obstacles. This is your portfolio piece.

07
The Summit

Legged Locomotion & Humanoids

FOC, BLDC motors, MPC/LQR control, ZMP, sim-to-real reinforcement learning.

Advanced Control Theory
▶
Brian Douglas — Full Control Theory Channel
Beyond PID: Bode plots, state space, LQR, Kalman filter integration with control. The best control theory channel on YouTube. Work through in order: transfer functions → state space → LQR → MPC.
YouTube
🎓
MIT 6.832 — Underactuated Robotics (Russ Tedrake)
The definitive academic course on legged robot control. Covers inverted pendulums, ZMP, MPC, trajectory optimization. Full lecture videos, problem sets, and the textbook are all free online. This is where humanoid locomotion lives.
Free Course
FOC & BLDC Motors
▶
SimpleFOC — YouTube + Library
Open-source FOC library with extensive documentation and tutorials. The best practical introduction to driving BLDC motors with field-oriented control. Runs on Arduino/ESP32/STM32.
YouTube
🌐
SimpleFOC Documentation
The theory section explains FOC math clearly before implementation. Paired with the YouTube channel, this is the most accessible FOC resource available.
Website
Simulation: Reinforcement Learning & Sim-to-Real
⚙
MuJoCo — DeepMind (Free)
The world's best open-source physics simulator for articulated robot bodies. Free since 2022 (DeepMind acquisition). Used by every major humanoid lab. Start with the official docs and tutorials, then the MuJoCo Menagerie (pre-built humanoid models).
Free Tool
⚙
NVIDIA Isaac Lab (Isaac Sim)
GPU-accelerated robot learning framework. Train locomotion policies 1000x faster than real time. Used by Figure, Agility, and every serious humanoid lab for sim-to-real transfer. Open source. Requires NVIDIA GPU.
Free Tool
⎇
Humanoid-Gym — RobotEra (GitHub)
Open-source RL framework for humanoid locomotion. Trains in Isaac Gym, validates in MuJoCo, deploys zero-shot to physical robots. Reference implementation with complete training pipeline and reward functions.
GitHub
⎇
Legged Gym — ETH Zurich RSL (GitHub)
The codebase behind "Learning to Walk in Minutes." Quadruped and biped RL training environments in Isaac Gym. The most cited open-source legged locomotion codebase. Study this code deeply.
GitHub
⎇
MuJoCo Menagerie — DeepMind (GitHub)
Collection of high-quality URDF/MJCF robot models: Unitree H1, Unitree G1, Boston Dynamics Spot, Shadow Hand, and more. Use these as your simulation bodies instead of building from scratch.
GitHub
Reinforcement Learning Foundations
📄
Reinforcement Learning — Sutton & Barto (Free PDF)
The canonical RL textbook. Free official PDF. Covers MDPs, Q-learning, policy gradients — the mathematical foundations behind every locomotion policy. Read Chapters 1-6 and 13 before touching Isaac Lab.
Free PDF
▶
DeepMind x UCL — Deep Reinforcement Learning (YouTube)
Full lecture series from DeepMind researchers. Covers deep RL, policy gradient methods, and model-based RL. This is the theory behind Optimus and Atlas's locomotion brains.
YouTube
Final Milestone

Design a 12-DOF quadruped in CAD. Import URDF to Isaac Lab. Train a walking policy via RL. Validate in MuJoCo. Build the physical robot with BLDC motors and SimpleFOC. Deploy the policy. When it walks — you've completed the roadmap.
